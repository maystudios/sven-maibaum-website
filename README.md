# Sven Maibaum Portfolio (Modern Rebuild)

Modern React + TypeScript rebuild of the existing Sven Maibaum portfolio.

## Requirements
- Node.js 18+

## Install
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```

## Tech Stack
- Vite + React + TypeScript
- Tailwind CSS
- React Router
